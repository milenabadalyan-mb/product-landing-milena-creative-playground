# CreativeStudio - Product Landing Page

A modern, fully responsive landing page for CreativeStudio - an AI-powered creative design platform. Built with vanilla HTML, CSS, and JavaScript featuring smooth animations, interactive elements, and a stunning Picsart-inspired color scheme.

## âœ¨ Features

### Core Features (v1)
- **Hero Section** - Eye-catching gradient background with product name, tagline, and CTA button
- **Product Showcase** - 4 feature cards highlighting key product capabilities
- **Benefits Section** - 3 main benefits with custom SVG icons
- **Testimonials** - 3 customer testimonials with star ratings and author info
- **Pricing Section** - 3 pricing tiers (Free, Pro, Enterprise) with featured plan highlight
- **Footer** - Social media links, navigation columns, and copyright notice

### Interactive Features
- **Smooth Scrolling** - Seamless navigation between sections with anchor links
- **Scroll Animations** - Elements fade in and slide up as you scroll
- **Parallax Effect** - Hero content moves dynamically during scroll
- **Navbar Effects** - Background changes from transparent to solid on scroll
- **Hover Interactions** - Cards, buttons, and images respond to user interaction
- **Mobile Menu** - Responsive hamburger menu for mobile devices
- **Ripple Effects** - Visual feedback on button clicks
- **Loading Animation** - Smooth page load transition

### Design Highlights
- **Picsart-Inspired Colors** - Purple (#7C3AED), Pink (#EC4899), Blue (#3B82F6) gradients
- **Semantic HTML** - Proper use of `<header>`, `<main>`, `<section>`, `<footer>` tags
- **Fully Responsive** - Mobile-first design that works on all screen sizes
- **Modern UI/UX** - Card-based layouts, gradient accents, and smooth transitions
- **Accessibility** - Proper ARIA labels and semantic structure
